# OpenClaw Configure

Expert-level CLI configuration skill for OpenClaw — covers 25+ domains, 142+ commands.
OpenClaw CLI 专家级配置技能，覆盖 25+ 配置领域、142+ 命令。

## Features / 功能

- **Channels** — Add and manage 20+ messaging platforms (Telegram, WhatsApp, Discord, Slack, iMessage, etc.) / 接入和管理 20+ 消息平台
- **Models** — Switch AI models, configure providers, set up OAuth2 auth / 切换 AI 模型、配置服务商、设置 OAuth2 认证
- **Plugins** — Enable/disable feature plugins, manage the plugin gate / 启用或禁用功能插件
- **Gateway** — Start, stop, restart, and monitor the OpenClaw gateway / 启停和监控 OpenClaw 网关
- **Agents** — Configure agents, hooks, cron jobs, and memory / 配置代理、钩子、定时任务和记忆
- **Security** — Sandbox, approvals, webhooks, and access control / 沙箱、审批流、Webhook 和访问控制
- **ClawHub** — Search, install, and publish skills via the skill registry / 通过技能市场搜索、安装和发布技能
- **Self-Evolving** — Auto-detects new OpenClaw versions and refreshes its own reference docs / 自动检测新版本并更新自身参考文档
